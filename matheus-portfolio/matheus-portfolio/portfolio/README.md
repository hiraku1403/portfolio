Este é o repositório do portfólio profissional de **Matheus Henn**, desenvolvido com tecnologias modernas de desenvolvimento web para apresentar projetos, habilidades, trajetórias profissionais e meios de contato de forma performática, responsiva e elegante.

O projeto está implantado e disponível para visualização na plataforma **Vercel**.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias e ferramentas:

- **[React.js](https://react.dev/)** / **[Next.js](https://nextjs.org/)** — Framework web para React com foco em performance e renderização otimizada.
- **[Tailwind CSS](https://tailwindcss.com/)** — Framework utilitário de CSS para um design responsivo, limpo e altamente customizável.
- **[TypeScript](https://www.typescriptlang.org/)** — Tipagem estática para garantir consistência e evitar erros em tempo de execução.
- **[Lucide React](https://lucide.dev/)** / **[FontAwesome](https://fontawesome.com/)** — Pacote de ícones modernos e minimalistas.
- **[Vercel](https://vercel.com/)** — Plataforma de nuvem para hospedagem rápida e integração contínua (CI/CD).

---

## ✨ Funcionalidades

- **Design Responsivo (Mobile-First):** Layout adaptável para smartphones, tablets e desktops.
- **Seção Sobre Mim:** Breve resumo profissional, objetivos e formação.
- **Portfólio de Projetos:** Galeria com os principais projetos desenvolvidos, contendo links para código fonte e demonstrações ao vivo.
- **Habilidades & Competências:** Exibição interativa de tecnologias dominadas (Front-end, Back-end, ferramentas, etc.).
- **Formulário de Contato / Links Sociais:** Integração para redes como LinkedIn, GitHub e envio de e-mails diretos.
- **Modo Escuro/Claro (Opcional):** Suporte visual para melhor legibilidade de acordo com a preferência do usuário.

Você precisará ter instalado em sua máquina o [Node.js](https://nodejs.org/) e um gerenciador de pacotes (npm, yarn ou pnpm).

### 1. Clonar o Repositório
Para clonar a pasta específica deste projeto usando Git:
